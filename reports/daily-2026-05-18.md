PowerShell에서 직접 실행 필요:
```powershell
cd C:\home\jeon\api-health-monitor
Remove-Item .git\index.lock -Force
Remove-Item .git\HEAD.lock -Force
Remove-Item .git\refs\heads\main.lock -Force
git reset HEAD
git status
```
이후 정상 git 사용 가능.
