# CheckAPI Logo Concepts

## 로고 시안 설명

### Concept 1: Checkmark + Pulse (logo-concept-1-checkmark.svg)
- **특징**: 체크마크와 맥박 라인 결합
- **색상**: 블루 (#3B82F6)
- **느낌**: 헬스 모니터링, 실시간 체크
- **용도**: 앱 아이콘, SNS 프로필

### Concept 2: Shield (logo-concept-2-shield.svg)
- **특징**: 방패 모양 + API 브라켓 + 체크마크
- **색상**: 그린 (#10B981)
- **느낌**: 보안, 안정성, 신뢰
- **용도**: 기업용 느낌, 보안 강조

### Concept 3: Minimal (logo-concept-3-minimal.svg)
- **특징**: 미니멀 라운드 스퀘어 + 심플 아이콘
- **색상**: 인디고 (#6366F1)
- **느낌**: 모던, 심플, 클린
- **용도**: 앱 아이콘, 파비콘

### Concept 4: Modern Gradient (logo-concept-4-modern.svg)
- **특징**: 그라디언트 + 코드 브라켓 + 상태 인디케이터
- **색상**: 퍼플-블루 그라디언트
- **느낌**: 현대적, 개발자 친화적
- **용도**: 랜딩 페이지, 마케팅

### Concept 5: Heartbeat Monitor (logo-concept-5-heartbeat.svg)
- **특징**: 모니터 화면 + 심전도 라인
- **색상**: 다크 + 그린 (#10B981)
- **느낌**: 실시간 모니터링, 기술적
- **용도**: 대시보드 헤더, 기술 문서

### Concept 6: Simple Wordmark (logo-concept-6-simple.svg)
- **특징**: 심플 아이콘 + 워드마크
- **색상**: 블루 (#3B82F6)
- **느낌**: 깔끔, 전문적
- **용도**: 헤더 로고, 이메일 서명

## 파일 용도별 추천

### 앱 아이콘 (512x512)
- **추천**: Concept 1 또는 Concept 3
- 정사각형 비율, 심플한 형태가 작은 사이즈에서도 인식 잘됨

### 웹사이트 헤더
- **추천**: Concept 6
- 가로로 길어서 네비게이션 바에 적합

### SNS 프로필
- **추천**: Concept 1, 2, 3
- 원형으로 크롭되어도 잘 보임

### Product Hunt
- **추천**: Concept 4 (tagline 포함)
- 설명이 함께 있어서 첫인상 효과적
