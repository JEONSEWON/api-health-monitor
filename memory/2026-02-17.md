## üìù Lessons Learned

### What Went Well
1. **FastAPI** - Extremely fast development, excellent docs
2. **Next.js 14** - App Router is powerful once understood
3. **Railway** - Easiest backend deployment ever
4. **LemonSqueezy** - Simpler than Stripe for Korea-based dev
5. **Tailwind CSS** - Rapid UI development
6. **OpenClaw** - AI-assisted development accelerated everything

### Challenges Overcome
1. **Celery + Beat** - Combined worker config tricky but working
2. **Webhook Signatures** - Raw body reading for HMAC verification
3. **User Management** - Finding user by email vs custom_data
4. **Modal Text Colors** - CSS specificity issues with Tailwind
5. **Favicon Generation** - Node.js sharp library solution

### What I'd Do Differently
1. Start with PostgreSQL instead of SQLite
2. Set up automated tests earlier
3. Create more reusable React components
4. Document API as I build, not after
5. Test email alerts earlier (SendGrid setup)
